Security Policy
Supported Versions

We provide security updates and fixes for the following versions:

Version	Supported
5.1.x	✅ Yes
5.0.x	❌ No
4.0.x	✅ Yes
< 4.0	❌ No

Only supported versions will receive security patches. Upgrade to the latest supported version if you want continued coverage.

Reporting a Vulnerability

If you discover a security issue, please report it responsibly:

Contact: Send details to security@yourdomain.com
 or open a private issue marked security.

Response Time: Expect an acknowledgment within 48 hours.

Resolution: We’ll work with you to verify, patch, and release a fix.

Public Disclosure: Please do not disclose publicly until the vulnerability is fixed.

We appreciate responsible disclosure and will credit you in release notes if you wish.
