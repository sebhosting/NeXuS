# Security Policy

## Supported Versions

We provide security updates and fixes for the following versions:

| Version  | Supported |
| -------- | --------- |
| 5.1.x    | ✅ Yes    |
| 5.0.x    | ❌ No     |
| 4.0.x    | ✅ Yes    |
| < 4.0    | ❌ No     |

> Only supported versions receive security patches. Upgrade to a supported version for continued coverage.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly:

- **Contact:** security@sebhosting.com  
- **Response Time:** We will acknowledge your report within **48 hours**.  
- **Resolution:** We will work with you to verify, patch, and release a fix.  
- **Disclosure:** Please **do not disclose publicly** until the vulnerability is resolved.  

We appreciate responsible disclosure and will credit you in release notes if you wish.
